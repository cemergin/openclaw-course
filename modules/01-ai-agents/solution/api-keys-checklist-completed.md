# API Keys Setup Checklist -- Completed Example

This is what a completed checklist looks like. Your specific details will differ.

## Claude (Anthropic) -- Required

- [x] Created account at console.anthropic.com
- [x] Added payment method (Plans & Billing)
- [x] Set monthly spending limit to $10
- [x] Generated API key named: openclaw-dev
- [x] Saved key in password manager
- [x] Key starts with: sk-ant- (verified, not truncated)

## OpenAI -- Optional

- [x] Created account at platform.openai.com
- [x] Added payment method / loaded $10 in credits
- [x] Set monthly budget limit to $10
- [x] Generated API key named: openclaw-dev
- [x] Saved key in password manager
- [x] Key starts with: sk- (verified, not truncated)

## Security Habits

- [x] I know where to revoke my Claude key (API Keys page, delete button)
- [x] I know where to revoke my OpenAI key (API Keys page, delete button)
- [x] My keys are NOT saved in plain text files, sticky notes, or chat messages
- [x] My keys are stored in: 1Password (Secure Notes > "AI API Keys")

## Billing Awareness

- [x] I checked my Claude usage dashboard (should be $0.00 or near it)
- [x] I understand the rough cost per message ($0.001 - $0.01 for simple messages)
- [x] I have a spending limit set so I won't get surprise-billed

## Notes

- Claude Sonnet seems like the best default for personal agent use
- Haiku is noticeably faster but less nuanced -- good for simple lookups
- Opus is impressive but 5x the cost -- save for important tasks
- Set a calendar reminder to check billing dashboard weekly during setup phase
