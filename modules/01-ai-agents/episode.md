# Episode 1: What Are AI Agents (And Why Run Your Own)?

## In This Episode

Before we touch a single server, we need to talk about *what* we're building and *why* it matters. You'll learn what separates a basic chatbot from a real AI agent, how OpenClaw's architecture actually works (spoiler: it's beautifully simple), and why running your own gives you powers that no hosted service can match. Then we'll get your API keys set up so you're ready to build.

## Key Concepts

- AI agents vs chatbots vs assistants -- what makes each one tick
- The OpenClaw architecture: your phone, a chat app, Docker, and AI APIs
- API keys and pay-per-use billing (how the money works)
- Self-hosting tradeoffs: control and privacy vs maintenance responsibility
- Claude vs OpenAI: choosing your AI brain

## Prerequisites

You should be comfortable using a web browser and creating online accounts. That's it -- this is the conceptual starting point.

**Self-check:** Can you sign up for a web service, navigate account settings, and copy-paste a long string without accidentally trimming characters? You're good.
