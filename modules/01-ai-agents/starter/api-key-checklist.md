# API Key Setup Checklist

Use this checklist to track your progress through the exercise.

## Anthropic (Claude) -- Required

- [ ] Created account at console.anthropic.com
- [ ] Added payment method (Plans & Billing)
- [ ] Set monthly spending limit ($10 recommended)
- [ ] Set notification threshold ($5 recommended)
- [ ] Generated API key (name: `openclaw-dev`)
- [ ] Saved key in password manager
- [ ] Verified key starts with `sk-ant-`
- [ ] Bookmarked the API Keys page (for emergency revocation)

## OpenAI (GPT) -- Optional

- [ ] Created account at platform.openai.com
- [ ] Added payment method / loaded credits ($10)
- [ ] Set monthly budget limit
- [ ] Generated API key (name: `openclaw-dev`)
- [ ] Saved key in password manager
- [ ] Verified key starts with `sk-`
- [ ] Bookmarked the API Keys page

## Security Habits

- [ ] Keys are stored in a password manager (not a text file on your desktop)
- [ ] Keys are NOT in any git repo, screenshot, or public chat
- [ ] I know where to revoke each key in under 30 seconds
- [ ] I have a spending limit set on every account
